# 💰 Controle de Gastos CLI

## 📌 Problema
Muitas pessoas têm dificuldade em controlar seus gastos diários, o que pode levar a desorganização financeira e dívidas.

## 💡 Solução
Esta aplicação em linha de comando permite registrar, visualizar e analisar gastos de forma simples e rápida, ajudando no controle financeiro pessoal.

## 👥 Público-alvo
- Estudantes
- Pessoas que desejam organizar suas finanças
- Usuários que preferem soluções simples e leves

## ⚙️ Funcionalidades
- Adicionar gastos com valor e categoria
- Listar todos os gastos registrados
- Calcular o total de gastos
- Filtrar gastos por categoria

## 🛠️ Tecnologias utilizadas
- Python
- Pytest (testes automatizados)
- Ruff (linting)
- GitHub Actions (CI)
